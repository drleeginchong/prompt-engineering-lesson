# 🧠 Learning Prompt Engineering

Welcome to my journey into the world of AI communication! 
This repository serves as a personal laboratory where I experiment with, document, and master the art of prompt engineering.

## 🚀 Learning Roadmap

### 🏁 Phase 1: Beginner (Foundations)
*Status: [In Progress]*
- [ ] **Core Elements:** Understanding Instructions, Context, and Output Indicators.
- [ ] **Role Prompting:** Assigning personas (e.g., "Act as a Python Mentor").
- [ ] **Zero-Shot vs. Few-Shot:** Comparing performance when providing 0 vs. 3 examples.
- [ ] **Delimiters:** Using `"""`, `---`, or `###` to separate instructions from data.

### ⚙️ Phase 2: Intermediate (Reasoning & Structure)
- [ ] **Chain of Thought (CoT):** Mastering the "Think step-by-step" technique.
- [ ] **Structured Outputs:** Forcing the model to return valid JSON or Markdown tables.
- [ ] **Prompt Chaining:** Breaking one complex task into three smaller, linked prompts.
- [ ] **RAG Basics:** Experimenting with how models handle provided external knowledge.
